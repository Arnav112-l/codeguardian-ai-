How It Works

/reset Step 1

Load Scenario

Agent calls /reset. Platform loads a real developer task from the repository and returns the initial observation to the agent.

/step Step 2

Agent Acts

Agent sends its structured decision — classify an issue, detect a vulnerability, or propose a complex dependency update.

grader Step 3

Grader Evaluates

Task-specific graders score the action on accuracy, detection rate, and reasoning quality using deterministic logic.

reward Step 4

Reward Signal

Platform provides detailed telemetry and reward signals back to the agent for reinforcement learning optimization.
